# 2026_spring_dpo_hse_cl

## Формула оценок

- **0.25** – тест в середине курса,
- **0.1** – мини-задание в середине курса,
- **0.65** – самостоятельный анализ с отрисовкой графика и проведением статистических тестов.

## План занятий

1. Введение в R и RStudio. Векторы
2. Приведение типов. Векторизация. Ресайклинг. Индексирование векторов
3. Логические векторы. NA, NaN, NULL. Матрицы и массивы.
4. Списки, датафреймы. Работа с пакетами.
5. Импорт и экспорт данных.
6. Создание функций, семейство apply(), data.table vs. tidyverse
7. tidyverse
8. tidyverse, Описательная статистика
9. Описательная статистика, визуализация в base R (+ тест на 20 минут)
10. {ggplot2} (+ мини-дз)
11. Quarto
12. Статистика вывода: введение
13. Статистика вывода: выборочное распределение средних, CLT, NHST
14. Статистика вывода: статистические тесты
15. Статистика вывода: статистические модели

16. Защита проектов

## Установка пакетов

Если из России не устанавливается пакет, то попробуйте:

```
chooseCRANmirror()
```

Выбираем сначала 73 (и жмем `enter`)
Потом 9 (сервер в Новосибирске)
Если не помогло, то попробуйте китайский севрер:

```
chooseCRANmirror()
```

Выбираем 17 (и жмем `enter`)
После этого запускаем `install.packages("название пакета")` заново.
